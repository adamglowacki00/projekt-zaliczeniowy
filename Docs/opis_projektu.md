# Opis Projektu

## Cel Aplikacji
Projekt dotyczy prostej aplikacji budżetowej dla studentów.

## Rozwiązywany Problem
Aplikacja rozwiązuje problem braku kontroli nad wydatkami na jedzenie i rozrywkę. Pomaga zapanować nad studenckim budżetem.

## Odbiorcy
Głównym odbiorcą są studenci, którzy chcą zaoszczędzić pieniądze na wakacje lub inne cele.

## Główne funkcje
* Dodawanie paragonów i wydatków
* Wykres wydatków (statystyki)
* Ustawienie limitu miesięcznego
